# AI-Driven Personalized Learning Framework for Schools and Universities

## Overview
This project is an AI-powered personalized learning management system designed to analyze student academic data and provide performance insights, predictions, and learning recommendations. The system helps teachers identify weak students early and supports data-driven academic decision-making.

The system supports three user roles:
- **Student**
- **Teacher**
- **Admin**

It analyzes academic assessments such as quizzes, assignments, and mid-semester exams to generate topic-wise analysis and predict future academic performance.

---

## Key Features

- Role-based dashboards (Student / Teacher / Admin)
- Topic-wise academic performance analysis
- Strong, average, and weak topic identification
- Academic performance prediction
- Personalized learning recommendations
- Excel/CSV upload for academic data
- Interactive charts and dashboards

---

## System Architecture

Users interact with the system through a React-based frontend.  
The frontend communicates with a Django REST API backend.  
The backend processes academic data using logic-based AI analysis and stores results in the database.

**Technology Flow:**

Users → React Frontend → Django REST API → AI Logic Processing → SQLite Database
